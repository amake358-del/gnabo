# Paiements

## Modes de paiement acceptés

| Mode | Détail |
|---|---|
| Espèces | Enregistrement en caisse |
| Carte bancaire | Terminal TPE |
| Chèque | Numéro + banque |
| Virement | Référence + date |
| Mobile Money | Orange Money, Wave, etc. |

## Enregistrement

- Lié à une facture (obligatoire)
- Montant (peut être partiel)
- Mode de paiement
- Référence (numéro de chèque, ref virement, etc.)
- Date
- Utilisateur qui a encaissé

## Paiements partiels

- Un devis/facture peut être payé en plusieurs fois
- Suivi du solde restant
- Affichage "Payé en X fois"

## Crédits

- Un crédit est un paiement différé
- Montant total dû
- Échéancier (optionnel)
- Statut : en cours, soldé, impayé
- Notification si échéance non respectée

## Reçu PDF

Généré automatiquement à chaque paiement :
- Numéro de reçu
- Date
- Client
- Montant
- Mode de paiement
- Reste à payer (si paiement partiel)
- Signature / cachet

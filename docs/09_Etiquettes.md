# Étiquettes

## Description

Étiquette autocollante apposée sur chaque appareil du module Électronique,
permettant une identification rapide sans avoir à scanner le QR Code.

## Contenu de l'étiquette

```
┌──────────────────────────┐
│  GNABO MULTI-SERVICES    │
│  ÉLECTRONIQUE            │
│                          │
│  ┌──────────────────┐    │
│  │     QR CODE      │    │
│  │    (UID hex)     │    │
│  └──────────────────┘    │
│                          │
│  EL-000001               │
│  Samsung Galaxy S23      │
│  Reçu: 15/07/2026        │
└──────────────────────────┘
```

## Format

- Taille : 60×40 mm (étiquette standard)
- Impression sur feuille A4 prédécoupée (10× par feuille, 2 colonnes × 5 lignes)
- Contraste élevé (noir sur blanc)
- QR Code lisible à 10 cm

## Génération

- Automatique à la réception d'un appareil
- Accessible depuis la fiche appareil
- Impression par lot (plusieurs étiquettes en une page)
- Ré-impression possible à tout moment

## Utilisation

- Collée sur l'appareil (ou sur le sachet contenant les accessoires)
- Permet de retrouver l'appareil dans l'atelier sans ouvrir le logiciel
- Le QR Code permet un scan rapide pour charger la fiche complète

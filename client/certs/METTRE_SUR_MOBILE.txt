INSTALLER LA CA mkcert SUR MOBILE
==================================

Le fichier `rootCA.pem` est le certificat de l'autorité de certification
locale. Une fois installé sur votre mobile, tous les certificats générés
par mkcert (pour n'importe quel projet) seront automatiquement trustés.

--- ANDROID ---

1. Transférez `rootCA.pem` sur le téléphone (mail, Drive, Bluetooth, USB)
2. Si nécessaire, renommez le fichier en `rootCA.crt`
3. Allez dans : Paramètres → Sécurité → Chiffrement et identifiants
   → Installer depuis le stockage → sélectionnez `rootCA.crt`
4. Donnez un nom (ex: "mkcert Dev CA"), validez
5. Ouvrez Chrome → https://192.168.13.101:5173/ → cert vert → PWA prêt

--- iOS (iPhone/iPad) ---

1. Envoyez-vous `rootCA.pem` par mail ou AirDrop
2. Ouvrez le fichier → "Profil téléchargé" apparaît
3. Allez dans : Réglages → Profil téléchargé → Installer (code PIN)
4. Allez dans : Réglages → Général → Informations → Trust des certificats
5. Activez le switch pour "mkcert"
6. Ouvrez Safari → https://192.168.13.101:5173/ → cert vert → PWA prêt
